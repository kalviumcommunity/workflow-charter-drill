# Contributing to Workflow Charter Drill

We welcome contributions! To maintain a high standard of engineering, please follow these steps:

1. **Fork the repository** and create your branch from `main`.
2. **Follow the Branching Strategy** defined in [BRANCHING.md](./BRANCHING.md).
3. **Commit with Discipline**: Use conventional commit messages.
4. **Submit a Pull Request**: Use the provided template and ensure all CI checks pass.
5. **Code Review**: All PRs require at least one approval from a code owner.

## Development Setup
```bash
npm install
npm start
```
